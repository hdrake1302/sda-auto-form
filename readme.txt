Tính năng SDA - Auto Form 4.0.0:
- Copy form đang làm (All)
- Copy form kết quả (GGForm + TDTUe)
- Copy với bỏ qua câu sai chỉ lấy câu đúng nếu có 
(Tùy chọn Ignore - GGForm)
- Copy đáp án nối tiếp (Tùy chọn Append+ - All)
- Copy key từ text được tạo bởi extension trên trang SDA - Share

- Nhập key code vào SDA-Auto Form (All)
- Paste đáp án từ keys đã được copy (All)
- Paste đáp án ngẫu nhiên với tùy chọn random 
(Câu nào không có trong key sẽ được tool random câu trả lời) (GGForm + MyAloha)

- Share đáp án với người khác bằng text hoặc key code (SDA - Share)

- Share đáp án thông qua trang SDA - Share 
(Ấn vào nút share trên extension)
